# Nikeeta Malik — Portfolio

A modern, interactive portfolio showcasing my work in full-stack development, UI/UX design, and cybersecurity fundamentals. Designed to reflect both technical capability and a strong focus on user-centered experiences.

## Live Preview

Coming soon (Netlify deployment in progress).

## Built With

- HTML5, CSS3, JavaScript  
- Typed.js for dynamic text animations  
- Custom UI effects (glitch, holographic interface, intro animations)  
- Font Awesome & Google Fonts  
- Netlify (planned for deployment and form handling)

## Features

- Interactive landing experience with animated intro
- Clean, responsive UI with modern design aesthetics
- Structured skills showcase across development, design, and security
- AI-powered avatar with voice interaction
- Integrated contact form with Netlify handling
